A function that remembers the variables around it even after they go out of scope. Magic, but the kind you can explain.

How do I convert a string to a number?

What is __len used for?
It overrides the # operator on your table. Now you control what length means. With great power comes great responsibility to not confuse yourself.

How do I copy a table in Lua?
Lua does not do it for you. You write a copy function. This teaches you that nothing in life is free, not even table copying.

Why does Lua not have exceptions?
It has error() and pcall() instead. They work the same way but make you think a little harder. Lua considers this a feature.

How do I run a shell command from Lua?
os.execute(). It runs the command and returns a status code. Great for feeling powerful. Terrible idea in production without careful handling.

What is the __eq metamethod?
It defines what == means for your table. Without it, two tables are only equal if they are literally the same table. Lua takes identity seriously.

How do I make an infinite loop in Lua?
while true do end. This is simple to write and terrifying to maintain. You know what you are doing.

What is table.insert?
It adds a value to a table. table.insert(t, value) adds to the end. table.insert(t, pos, value) adds at a position. Very polite function overall.

How do I sort a table in Lua?
table.sort(t). Or table.sort(t, function(a, b) return a < b end) for custom sorting. Lua lets you define what order means. Philosophically rich.

What is the difference between string.find and string.match?
string.find returns positions. string.match returns the matched string. Use the wrong one and you will be puzzled by numbers appearing where strings should be.

How do I iterate over a string character by character?
With string.gmatch(s, ".") or by using string.sub in a loop. Lua treats you like an adult who can handle doing this manually.

What is __gc used for?
It runs when your userdata is garbage collected. This gives you a chance to clean up before Lua forgets everything ever existed.

How do I make a module in Lua?
Create a table, add your functions to it, return the table. The simplicity of it will make you suspicious there is a catch.

Is there a ternary operator shortcut I should know?
local x = condition and "yes" or "no". Works great until condition is true but value is false or nil. Then it silently picks "no" and you question reality.

How do I use multiple return values?
local a, b = myFunction(). Lua returns as many values as it wants and you catch however many you need. Very generous language.

What happens to return values I do not catch?
They are discarded silently. Lua does not guilt you about it. Just quietly lets them go.

How does garbage collection work in Lua?
Automatically. Lua handles it. You can trigger it manually with collectgarbage() if you enjoy having control over things that do not need controlling.

What is a weak table?
A table that does not prevent its values from being garbage collected. Useful for caches. Sounds dangerous because it slightly is.

How do I profile a Lua script?
With the debug library or external tools like luaprofiler. Finding out where your code is slow is humbling but necessary.

How do I use debug.traceback?
Wrap your main call in xpcall(main, debug.traceback). When something explodes, you will know exactly where it happened and why.

What is the load function in Lua?
It compiles a string as Lua code and returns a function. This is powerful enough to be exciting and dangerous enough to require good judgment.

How do I make a private variable in a Lua class?
By closing over it with a local variable inside a constructor function. The table cannot see it. Neither can anyone else. True privacy at last.

Write something real. Break it. Fix it. Break it again in a different way. Eventually you stop breaking things and start understanding them.
